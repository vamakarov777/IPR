

bootstrap - скрипты создания первичной структуры БД
conf - конфигурационные файлы: настройки подключения, параметры для выполнения скриптов
create_db - создание экземпляра БД
create_user - создания пользователя
lib - библиотеки, необходимые для выполения скриптов обслуживания БД

db_create.bat - запуск создания пользователя и БД
